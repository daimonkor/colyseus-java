# Colyseus-Kotlin

Implementation of Colyseus client using Kotlin

## Download

Maven:

```xml
<dependency>
    <groupId>ir.doorbash</groupId>
    <artifactId>colyseus-java</artifactId>
    <version>2.4</version>
    <type>pom</type>
</dependency>
```

Gradle: 
```groovy
dependencies {
    implementation 'ir.doorbash:colyseus-java:2.4'
}
```
